# 효과음 파일 목록

이 폴더에는 게임에서 사용되는 효과음 파일들이 들어갑니다.

## 필요한 효과음 파일 (.wav 형식 권장)

### 플레이어 액션
- `jump.wav` - 점프 효과음
- `dash.wav` - 대시 효과음
- `attack.wav` - 근접 공격 효과음
- `damage.wav` - 플레이어 피격 효과음
- `death.wav` - 플레이어 사망 효과음
- `projectile.wav` - 원거리 공격 발사 효과음

### 적 관련
- `enemy_death.wav` - 일반 적 처치 효과음

### 보스 관련
- `boss_hit.wav` - 보스 피격 효과음
- `boss_attack.wav` - 보스 공격 효과음

### 환경 및 효과
- `explosion.wav` - 폭발 효과음 (화염구 충돌)

### 아이템 및 시스템
- `item_collect.wav` - 아이템 획득 효과음
- `chest_open.wav` - 상자 열기 효과음
- `checkpoint.wav` - 체크포인트 활성화 효과음
- `victory.wav` - 승리 효과음

## 참고사항
- 파일이 없어도 게임은 정상적으로 실행됩니다 (효과음만 재생되지 않음)
- .wav, .ogg, .mp3 형식 모두 사용 가능
- 무료 효과음 리소스: freesound.org, opengameart.org
