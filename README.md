# Darkspire

2D 횡스크롤 플랫포머 액션 게임

## 실행 방법

```bash
python3 main.py
```

## 조작법

### 이동

- 방향키 (←/→) 또는 A/D: 좌우 이동
- Space 또는 ↑: 점프

### 전투

- F: 대시 (쿨타임 2초, 무적 0.2초)
- Z: 근접 공격 (검 필요)
- X: 원거리 공격

### 메뉴

- ↑/↓: 메뉴 선택
- Enter: 선택
- R: 재시작 (게임오버/승리 시)

## 게임 규칙

- 체력: 하트 3개
- 하트가 모두 소진되면 1스테이지부터 재시작
- 3개 스테이지 클리어 시 승리
